L'input n1 et n2 sont des "box" où l'utilisateur écrit du texte, dans la première box sera stocké le nom de l'utilisateur, tant dis que, dans la deuxième, ça sera le message que l'utilisateur souhaite envoyé. Le JS utilise d'ailleurs, la concaténation de deux parties de code JS, permettant de faire succéder plusieurs actions au même moment
